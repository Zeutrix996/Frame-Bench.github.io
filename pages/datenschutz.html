3. Datenerfassung auf dieser Website

FrameBench erfasst ausschließlich technische Daten, die für die Bereitstellung der Dienste erforderlich sind. Dazu gehören:

  • Browser-Informationen (User-Agent, Browserversion)
  • Hardware-Daten via Browser-APIs (siehe Abschnitt 4)
  • IP-Adresse (bei Server-Logs, falls vorhanden)
  • Referrer-URL (Herkunftsseite)
  • Zeitstempel der Anfrage

Es werden keine persönlichen Identifikationsmerkmale wie Name, Adresse oder E-Mail gespeichert, es sei denn, Sie geben diese freiwillig an (z.B. per Kontaktformular).

4. Nutzung von Browser-APIs

FrameBench nutzt folgende Browser-APIs zur Datenerfassung. Diese Daten werden lokal im Browser verarbeitet und nicht an Server übertragen:

  • navigator.hardwareConcurrency (Anzahl der CPU-Kerne)
  • navigator.deviceMemory (RAM-Schätzung in GB)
  • performance.memory (Browser-Speicher)
  • navigator.getBattery() (Batterie-Status, falls unterstützt)
  • navigator.connection (Netzwerk-Informationen)
  • Performance API (FPS-Messung und Performance-Analyse)

Rechtsgrundlage: Art. 6 Abs. 1 lit. a DSGVO (Einwilligung) durch Nutzung der Website.

5. Cookies und LocalStorage

Unsere Website verwendet keine Cookies zur Speicherung personenbezogener Daten. Es werden ausschließlich technische Session-Cookies für die Funktionalität der Website verwendet, falls erforderlich.

Der LocalStorage wird ausschließlich für die Speicherung von GPU-Bewertungen verwendet, falls Sie diese Funktion nutzen. Diese Daten verbleiben lokal auf Ihrem Gerät und werden nicht an Server übertragen.
